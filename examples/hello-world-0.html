HelloWorld

HelloWorld.swift:

open your Xcode Project and create a new swift file in ‘Sources’. Name it HelloWorld.swift and paste the following:

//HelloWorld.swift
import Foundation

class HelloWorld {
	
	init() {
	}
	
	func send_string(string: String) {
		print(string)
	}
}

hello_world.py

now in finder goto your project folder and locate ‘wrapper_sources’ and create hello_world.py. Paste the following:

#hello_world.py
from swift_tools.swift_types import *

@wrapper
class HelloWorld:

    def send_string(self, string: str): ...

(this is the same swift class represented in a wrapper.py file)
